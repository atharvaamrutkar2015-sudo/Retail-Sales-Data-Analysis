# Retail-Sales-Data-Analysis
Retail sales data analysis using Python, NumPy, Pandas and Matplotlib
# Retail Sales Data Analysis using Python

## Project Overview

This project analyzes a retail sales dataset to uncover meaningful business insights.
Using Python libraries such as **NumPy, Pandas, and Matplotlib**, the project performs data preprocessing, exploratory data analysis (EDA), and visualization to understand sales patterns and trends.

The analysis helps identify high-performing products, regional sales distribution, and monthly sales trends that can support business decision-making.

---

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Google Colab

---

## Dataset

The dataset contains retail sales transaction data with the following columns:

| Column   | Description                          |
| -------- | ------------------------------------ |
| Date     | Date of purchase                     |
| Product  | Product name                         |
| Category | Product category                     |
| Region   | Sales region                         |
| Quantity | Number of units sold                 |
| Price    | Price per unit                       |
| Sales    | Total sales value (Price × Quantity) |

---

## Project Objectives

The main objectives of this project are:

* Load and preprocess retail sales data
* Calculate total revenue generated from sales
* Identify the highest revenue product categories
* Determine which region contributes the most to sales
* Analyze monthly sales trends over time
* Identify the top 5 best-selling products
* Visualize the results using charts

---

## Data Analysis Tasks

### 1. Total Revenue

Calculate the total revenue generated from all sales transactions.

### 2. Sales by Product Category

Analyze which product categories generate the highest revenue.

### 3. Regional Sales Distribution

Determine which region contributes the most to overall sales.

### 4. Monthly Sales Trend

Analyze how sales change over time on a monthly basis.

### 5. Top 5 Best-Selling Products

Identify the products with the highest total sales.

---

## Visualizations

The following visualizations were created using **Matplotlib**:

* Monthly Sales Trend (Line Chart)
* Sales by Product Category (Bar Chart)
* Regional Sales Distribution (Pie Chart)
* Top 5 Best-Selling Products (Bar Chart)

These visualizations help better understand sales patterns and business performance.

---

## Key Insights

* Tablet is the highest selling product.
* Laptop and Smartwatch also generate strong sales.
* Sales vary across different regions.
* Monthly sales trends show fluctuations throughout the year.
* Certain months experience higher sales, indicating possible seasonal demand.

---

## Conclusion

This project demonstrates how Python can be used to analyze and visualize retail sales data effectively.
By performing exploratory data analysis and visualization, businesses can gain valuable insights into sales performance and customer demand patterns.
